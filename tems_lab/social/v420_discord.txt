hey everyone! v4.2.0 just dropped

Tem can now find bugs in itself and file GitHub issues about them. no joke.

it's called Tem Vigil — during idle time, Tem scans its own logs, uses its LLM to figure out if something is a real bug, scrubs all credentials out, and creates a GitHub issue. you just approve it.

set it up in 2 commands:
/addkey github — paste a PAT
/vigil auto — done

first real issue filed by Tem: https://github.com/temm1e-labs/temm1e/issues/28
